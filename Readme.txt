# Análisis de Ventas con NumPy y Pandas

## Descripción
En este ejercicio se crea un arreglo de NumPy a partir de una columna de ventas de un DataFrame y se realizan operaciones estadísticas básicas.

## Operaciones realizadas

- **np.mean()**  
  Calcula el promedio de los valores del arreglo de ventas.

- **np.max()**  
  Obtiene el valor máximo registrado en las ventas.

- **np.min()**  
  Obtiene el valor mínimo registrado en las ventas.

Estas operaciones permiten analizar rápidamente el comportamiento general de los datos.
